# Space game

A space-themed arcade game where you collect gold coins while avoiding meteors.

## How to Play
- Move your ship around the screen
- Collect gold coins to increase your score
- Avoid meteors — hitting one costs a 5 lives
- Game ends when you run out of lives

## Features
- Score and lives tracking (shown on screen)
- Randomly placed coins and meteors
- Simple space background 

## How to Run
Make sure you have Python installed (no extra libraries needed — this uses only built-in modules: `turtle`, `math`, and `random`).

```bash
python space_game.py
```

Just replace `space_game.py` with whatever your actual file is named.

## What I Learned
